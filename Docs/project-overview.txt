Project Title: IEEE WIE Static Website

Overview:
This project is a simple static website created for the IEEE Women in Engineering (WIE) student branch. 
The purpose of the website is to provide basic information about the WIE chapter, including its mission, 
team members, and contact details.

Objectives:
- To present the WIE chapter in a clear and organized manner.
- To provide visitors with an introduction to WIE and its goals.
- To display the leadership team and their roles.
- To offer a simple way for visitors to contact the chapter.

Technologies Used:
- HTML (for structure)
- CSS (for design and layout)
- JavaScript (for basic interactivity, if required)

Project Structure:
- Docs: Contains documentation and text content for the website.
- Designs: Contains design files, images, or graphics.
- Webpages: Contains HTML, CSS, and JavaScript files.

Status:
This is a basic-level project designed to help practice website structure, content organization, 
and simple design skills.
